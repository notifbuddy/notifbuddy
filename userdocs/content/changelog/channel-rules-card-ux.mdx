---
title: Channel rules — clearer save, safer delete
description: The Save action moved into each config card's header, and removing a config now asks for confirmation with wording that matches what's at stake.
date: 2026-07-19
tags: [dashboard]
---

Two quality-of-life fixes on the Channel rules page:

- **Save where you can see it.** Each team's config card now keeps its Save button in the header, next to the card's other actions — no more scrolling past the test panel to find it at the bottom of a long card.
- **Confirmation before removal.** Deleting a config asks first, and the dialog tells you what actually happens: removing a saved config stops new issues from opening Slack channels (existing channels stay untouched), while discarding a never-saved draft is called exactly that — a discard.
