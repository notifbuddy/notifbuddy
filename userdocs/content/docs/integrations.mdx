title: Integrations
description: Linear and Slack.
---

| Provider | Does |
| --- | --- |
| **Linear** | Opens a channel per issue; mirrors comments both ways; syncs team statuses. |
| **Slack** | Hosts the channels; syncs members for auto-add. |

## Channel rules

Each Linear team gets one config: how channels open, the name template, an optional condition, and members to auto-add. Configure these on the dashboard.
